##  Css: **BS4_Image**
---


* Description

Create responsive images by adding an .img-fluid class to the <img> tag. The image will then scale nicely to the parent element.



* Syntax

```html
<img class="img-fluid" src="img_chania.jpg" alt="Chania">

```
